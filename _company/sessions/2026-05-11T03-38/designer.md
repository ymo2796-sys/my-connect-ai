# 🎨 Designer — 영상 샘플의 썸네일 및 핵심 프레임(0초, 5초, 12초) 비주얼 브리프 작성. 고대비 컬러(노란색 진공 헤드 + 어두운 배경), 중앙 피사체 강조 구도, 모바일 최적화 비율(9:16) 적용. 텍스트 오버레이 위치와 폰트 가중치 지정.

🎨 Designer: 작업 시작합니다.

# 📐 무접촉 진공 청소기 쇼츠 — 비주얼 브리프 & 썸네일 가이드

## 1. 비주얼 시스템 (Visual System)
| 항목 | 명세 |
|:---|:---|
| **해상도/비율** | `1080 × 1920 px` (9:16 모바일 최적화) |
| **주요 컬러** | 진공 헤드/강점 강조: `#FFD600` (고대비 옐로우) → `#FF9100` (그라데이션 하이라이트) |
| **배경 컬러** | `#0A0A0A` (매트 블랙) → `#1A1A1A` (방사형 그라데이션, 중앙 밝게/외곽 어둡게) |
| **텍스트 컬러** | 메인: `#FFFFFF` (가독성 최우선) / 서브/아이콘: `#E0E0E0` |
| **타이포그래피** | `Pretendard` (가독성/모바일 최적화) / 대체: `Noto Sans KR` |
| **폰트 가중치** | 메인 문구: `ExtraBold (800)` / 서브/설명: `Bold (700)` |
| **자간/행간** | 자간 `-1%` ~ `0%` / 행간 `1.1` |

## 2. 모바일 안전지대 (Safe Area)
쇼츠/TikTok UI(좋아요, 프로필, 댓글창, 진행바)와 겹치지 않는 영역입니다.
- **수평 안전선:** `X: 108 ~ 972` (양측 여백 10%)
- **수직 안전선:** `Y: 250 ~ 1600` (상단 프로필/시간 13%, 하단 UI/CTA 17% 제외)
- **피사체 배치:** 모든 프레임에서 피사체 중심을 `X:540, Y:960` 기준으로 고정.

## 3. 썸네일 구도 명세 (Thumbnail Composition)
| 요소 | 사양 |
|:---|:---|
| **구도** | 중앙 피사체 강조 (Center-Focus). 진공 헤드 클로즈업이 프레임의 45% 차지 |
| **배경 처리** | `Gaussian Blur 12px` 적용한 어두운 작업 공간. 피사체와 배경의 채도/명도 대비 확보 |
| **조명/하이라이트** | 헤드 상단/전면에 `#FFD600` 에지라이트(폭 3px, 불투명도 80%) 적용. 먼지 입자에 `Rim Light` 반사 강조 |
| **텍스트 오버레이** | 위치: `X:140~940, Y:320~420` (상단 25% 지점, 프로필과 분리)<br>문구: `손 안 대고 끝!` (폰트 800, 64pt)<br>효과: `Drop Shadow` (색상 #000000 60%, X:0 Y:4, Blur:10) |
| **썸네일 전용 요소** | 우측 상단 `⚡` 아이콘 (Size: 48px, Color: #FFD600) 클릭 유도 |

## 4. 핵심 프레임 비주얼 가이드 (Key Frames)

### 🔹 0초 (Hook)
- **구도:** 극단적 클로즈업. 진공 입구(노란색 헤드)가 화면 중앙 하단(Y:1100)에 위치.
- **동작/효과:** 먼지 입자가 공기 중으로 부유하다 입구로 빨려 들어가는 순간 정지. 입자 에지에 `#FFD600` 글로우(Blur:6) 적용.
- **텍스트 오버레이:** `Y:380~480` 영역. `문장 8자 이내` 고정. 폰트 가중치 `800`. 배경에 `#000000` 40% 불투명도 사각형 마스크 적용하여 가독성 확보.

### 🔹 5초 (Core)
- **구도:** 제품 3/4 앵글. 집진통 분리 상태 노출. 피사체 중심 `X:540, Y:900`.
- **조명:** 배경은 `#0A0A0A` 유지. 제품 실루엣에 `#FFD600` 반사광(폭 5px)이 헤드와 분리 부위에만 적용.
- **텍스트 오버레이:** `Y:850~950` (하단 안전지대 상단). `무접촉 설계 / 일회용 위생` 분할 배치. 폰트 가중치 `700`. 자막 배경은 `#1A1A1A` 반투명 박스(Radius:8)로 UI 충돌 방지.

### 🔹 12초 (Result/CTA)
- **구도:** 왼쪽(먼지 집진통) vs 오른쪽(깨끗한 바닥) 비교 구도. 중앙에 제품 정면 배치.
- **효과:** 화면 하단에 `▶ 구매 링크` 화살표 애니메이션. 화살표는 `#FFD600` → `#FFFFFF` 그라데이션.
- **텍스트 오버레이:** `Y:1450~1550` (하단 15% 안전지대 바로 위). `지금 클릭해 구매 진행하세요` (폰트 800, 56pt). 배경에 `#000000` 70% 그라데이션 마스크 적용.

## 5. 텍스트 오버레이 전역 가이드
| 항목 | 지정 사항 |
|:---|:---|
| **위치 규칙** | 상단 UI(0~13%) / 하단 UI(83~100%) 절대 금지. 안전지대 `Y:250~1600` 내 배치 |
| **폰트 가중치** | 강조문구: `ExtraBold (800)` / 설명문구: `Bold (700)` |
| **가독성 확보** | 모든 텍스트에 `Outer Glow` (색상 #000000, 불투명도 50%, Blur:8) 적용. 배경이 어두워도 노란색 피사체와 흰색 텍스트 대비 유지 |
| **폰트 크기 기준** | 1080p 기준 메인: `56~64pt` / 서브: `40~48pt` (모바일 화면 기준 80% 이상 가독성 검증) |
