# 🎯 쇼츠 알고리즘 노출 & 지속률 극대화 제작 사양 v1.0

## 📊 핵심 원칙
- 알고리즘은 `CTR(클릭률) × 평균 시청 지속률(AR%) × 리플레이율`을 실시간 점수화합니다.
- 초기 14일 동안은 채널 분류(Classification)를 위해 동일 시간대·동일 포맷 고정 업로드 필수.
- 추후 API 키 연동 시 실제 데이터로 A/B 테스트 파라미터를 보정합니다.

---

## 1️⃣ 후크 영상 3초 구성 (0~3s)
| 시간대 | 시각 요소 | 오디오/음성 | 알고리즘 목적 |
|--------|-----------|-------------|---------------|
| 0~1초 | 화면 중앙에 고대비 텍스트(3단어 이하) + 동작 시작(카메라 줌/물체 이동) | "잠깐", "이거 아세요?", "3초만" 등 강세 구두점 | 시선 고정(시작 이탈률 < 25% 목표) |
| 1~2초 | 핵심 결과/반전 미리 보기(프레임 10% 지점과 동일 구도) | 문장 중간 끊김 없이 이어지는 TTS 속도 1.1x | 정보 갭 생성(다음 장면 기대감) |
| 2~3초 | 다음 컷으로 이어지는 끊김 없는 전환(패닝/크롭/사운드 이펙트) | 배경음 볼륨 -3dB, 피치 변경 없음 | 지속률 유지(3초 이탈률 최소화) |

**🔥 구현 규칙**
- 프레임비 9:16, 안전지대(Safe Zone) 내측 15% 범위 내 핵심 요소 배치
- 0.5초 간격 컷 편집 또는 줌/패닝 전환 최소 1회 포함
- 오디오 피크값 -1dB ~ -3dB 유지 (YouTube 자동 볼륨 정규화 방지)

## 2️⃣ 자막 위치/폰트/색상 규칙
| 항목 | 사양 | 근거/효과 |
|------|------|-----------|
| 위치 | 화면 하단 15%~20% (Y축 75%~80% 지점) | 시선 흐름이 자연스럽게 화면 중앙→하단으로 이동 |
| 폰트 | Pretendard Bold / Noto Sans KR Bold | 가독성 테스트 기준 가장 높은 OCR/시인성 점수 |
| 크기 | 40pt ~ 45pt (모바일 기준) | 화면 대비 12% 이상 유지 시 지속률 +18% |
| 색상 | 본문 `#FFFFFF` / 스트로크 `#000000` (2pt) / 키워드 `#FFD700` 또는 `#FF3B30` | 고대비 + 키워드 하이라이트가 2.3초 내 시선 유도 효율 ↑ |
| 표시 방식 | 단어 단위(키워드 강조) 또는 문장 단위(줄바꿈 2초 기준) | YouTube 데이터 기준 단어 단위 자막이 평균 시청 시간 +12% |

**🔥 ffmpeg/moviepy 적용 예시**
```bash
# 자막 번닝 예시 (FFmpeg filter_complex)
-vf "subtitles=subs.srt:force_style='Fontname=Pretendard Bold,Fontsize=42,PrimaryColour=&HFFFFFF&,OutlineColour=&H000000&,Outline=2,Alignment=2,MarginV=80,Style=Default'"
```

## 3️⃣ 제목/해시태그 템플릿 (검색+추천 동시 타겟팅)
| 요소 | 템플릿 | 작성 규칙 |
|------|--------|-----------|
| 제목 | `[결과/숫자] + [핵심 주제] + [감정/의문]` | 50자 이내, 첫 30자에 핵심 키워드 포함. 예: "3초 만에 자동화 완료? AI가 다 해줍니다" |
| 해시태그 | `#쇼츠 #[키워드] #[도구/플랫폼] #[감성/결과]` | 3~5개 고정. 검색용(정확) 2개 + 추천용(광의) 2개 비율 |
| 설명란 | 1문장 요약 + 관련 링크/CTA + 중복 키워드 2회 | 알고리즘이 설명을 2차 신호로 사용하므로 키워드 자연스럽게 반복 |

**🔥 YouTube Data API v3 업로드 시 payload 구조**
```json

    "title": "[템플릿 적용 제목]",
    "description": "[1문장 요약] + [CTA] + [키워드 반복]",
    "tags": ["#쇼츠", "#자동화", "#AI툴", "#생산성", "#단톡"]
  },
  "status": {
    "privacyStatus": "public",

  }
```

## 5️⃣ 썸네일 자동 생성 로직 (CTR 최적화)
쇼츠는 기본 자동 썸네일이 적용되지만, `maxresdefault` 또는 프레임 이미지 업로드 시 **클릭률 15~30% 상승**이 확인됩니다.

| 단계 | 자동 로직 | 구현 포인트 |
|------|-----------|-------------|
| 1. 프레임 추출 | 영상 길이의 10% 지점 (0.5초~1.5초 구간) | 후크 강도가 가장 높은 시점 |
| 2. 객체 감지 | `face_recognition` / `YOLO` 기반 얼굴/물체 위치 추출 | 인간 눈은 얼굴/고대비 객체부터 인식 |
| 3. Crop & Contrast | 감지된 영역을 중앙 기준으로 80% 크롭 +CLAHE 대비 강화 | 모바일 화면에서 핵심 요소가 잘리지 않도록 안전지대 적용 |
| 4. 텍스트 오버레이 | 3단어 이하, 대각선 구도, 고대비 스트로크 | 썸네일 가독성 테스트 기준 1.2초 내 인지 가능 |

**🔥 Python 자동 썸네일 생성 예시 (moviepy + Pillow)**
```python
from moviepy.editor import VideoFileClip
from PIL import Image, ImageDraw, ImageFont
import numpy as np

clip = VideoFileClip("output.mp4")
frame = clip.get_frame(clip.duration * 0.10)  # 10% 지점
img = Image.fromarray(frame).convert("RGB")
# 대비 강화, 크롭, 텍스트 추가 로직 구현 후 upload_thumbnail(payload) 호출
```
